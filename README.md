# Design and Analysis of Algorithms

This repository contains laboratory works for the course Design and Analysis of Algorithms.

## 📁 Structure

Each folder in this repository corresponds to a separate laboratory work:

- lab01-* — Laboratory Work 1  
- lab02-* — Laboratory Work 2  
- lab03-* — Laboratory Work 3  
- lab04-* — Laboratory Work 4  
- lab05-* — Laboratory Work 5  

## 🧪 Description

Each laboratory work includes:
- Source code written in C#
- Implementation of algorithms
- Practical tasks and experiments

## ⚙️ Technologies

- C#
- .NET

## 📌 Notes

Each project can be opened and run independently.
